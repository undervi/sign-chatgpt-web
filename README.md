  ## 🤖 기능 소개

  1. 수어 이미지 번역 
  - 손동작 이미지를 텍스트로 번역하여 의사소통의 장벽을 허물어줍니다.
  - Tensorflow Keras 딥러닝 모델 활용 (CNN)
  
  2. ChatGPT와의 대화
  - 번역된 문장은 ChatGPT API를 활용하여, 사용자에게 맞춤 대화를 제공합니다. 자동으로 생성된 답변은 사용자의 질문이나 입력에 유연하게 반응하여 자연스러운 대화를 이끌어냅니다.
